# Lucas's Agent Skills

Reusable [Agent Skills](https://agentskills.io) for AI coding agents. Compatible with Claude Code, Cursor, Codex, Gemini, Copilot, and any tool that follows the Agent Skills standard.

## Install

```bash
npx skills add LucasGois1/skills/<skill-name>
```

## Skills

| Name | Description |
|------|-------------|
| [mutation-hardening](skills/mutation-hardening/) | Java mutation testing loop with PITest. Configures, runs, parses survived mutants, writes targeted tests, loops until target met. |
